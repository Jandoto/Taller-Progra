Todo el programa almacena las matrices en tres listas y de formas distintas, una almacena el valor número, otra el valor alfabético y la otra el valor de las dos unidas.

El programa permite la lectura desde el archivo entrada.in donde los valores solo pueden ser de 1 dígito.
Por ejemplo, solo permite el ingreso de 1 no de 12 o de a y no de aa.

El cálculo del determinante y la inversa para matrices numéricas.

El programa permite múltiples cálculos de una misma matriz.

La implementación extra es el 5 del menu donde se pueden guardar las matrices en un Excel, cada hoja representa una matriz.
